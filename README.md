# Digital-Literacy-Portfolio

## Overview
This project consists of five tasks designed to develop digital literacy, professional communication, collaboration skills, and cyber awareness. Each task focuses on practical learning that helps students become responsible and effective digital users.

##  Project Structure

###  Task 1 – Digital Literacy Awareness Infographic
- Created a one-page infographic using a design tool (e.g., Canva)  
- Covered key topics such as:
  - Meaning of digital literacy  
  - Safe internet practices  
  - Useful digital tools / professional presence  
- Final design exported and stored in:  
  **task-1-presentation/**

###  Task 2 – Student Digital Portfolio
- Created/updated profiles on platforms like:
  - GitHub  
  - LinkedIn  
  - (Optional: Kaggle / Stack Overflow / ResearchGate)  
- GitHub profile README includes:
  - Name, branch, year, and learning goals  
- LinkedIn profile updated with education details  
- Screenshots saved in:  
  **task-2-portfolio/**

###  Task 3 – Coding & Collaboration Platforms

#### Part A – Coding Practice
- Completed a beginner-level challenge on platforms like:
  - HackerRank / CodeChef / HackerEarth  
- Screenshot of solution and score saved in:  
  **task-3-platforms/**

#### Part B – Google Form Quiz
- Created a Digital Literacy Awareness Quiz with 5 questions  
- Included:
  - Multiple choice questions  
  - Short answer questions  
- Added:
  - Form screenshot  
  - Linked Google Sheet (responses)  
- Files stored in:  
  **task-3-platforms/**


###  Task 4 – Professional Email & Etiquette
- Drafted two professional emails:
  - Request for assignment extension  
  - Internship application email  
- Followed proper format:
  - Clear subject line  
  - Professional greeting and closing  
  - Structured body  
- Created a Social Media Do’s and Don’ts checklist  
- Files stored in:  
  **task-4-email-etiquette/**


###  Task 5 – Cybercrime Awareness
- Developed a case study on Phishing:
  - Explained how the attack works step-by-step  
  - Identified targets and consequences  
- Created a Stay Safe Online checklist including:
  - General safety tips  
  - UPI/financial safety tips  
- Included reporting details:
  - National Cyber Crime Portal: https://cybercrime.gov.in  
  - Helpline: 1930  
- Files stored in:  
  **task-5-cybercrime/**


##  Objectives
- Enhance digital literacy and awareness  
- Build professional communication skills  
- Develop a strong digital portfolio  
- Understand and prevent cyber threats  
- Promote responsible use of digital platforms  


##  Key Learnings
- Digital literacy is essential for academic and career growth  
- Professional communication improves opportunities  
- Online platforms can be used effectively for learning and collaboration  
- Cyber awareness helps prevent financial and data loss  
- Responsible online behavior is crucial in today’s digital world  


##  Cyber Safety Resources
- National Cyber Crime Portal: https://cybercrime.gov.in  
- Helpline Number: 1930  


##  Conclusion
This project provided hands-on experience in digital tools, communication, and cybersecurity awareness. It strengthened essential skills required for academic success and future professional growth while promoting responsible digital citizenship.
